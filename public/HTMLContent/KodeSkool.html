Kode Skool

An innovative startup empowering school students with coding skills through hands-on lessons and real-world projects.

Project Overview

As a Full-Stack Developer, I built a comprehensive platform that enables students to learn coding through interactive lessons and practical projects. The platform features a modern, responsive design and robust backend infrastructure.

Technology Stack

JavaScript React.js Node.js MongoDB Bootstrap

Key Features

Interactive Dashboard

A personalized dashboard where students can track their progress, manage enrolled courses, and access learning resources.

Code Editor

An integrated code editor with real-time compilation and validation, providing immediate feedback to students.

AI-Powered Chatbot

An intelligent chatbot that provides instant support and guidance to students throughout their learning journey.

Student Profile

A comprehensive profile page where students can manage their personal information and view their achievements.

Backend Implementation

RESTful APIs

Developed comprehensive APIs for student data management, course handling, and user authentication.

Database Management

Implemented MongoDB for efficient data storage and retrieval with optimized queries.

Security

Integrated JWT and bcrypt for secure authentication, with email verification via Nodemailer.

AI Integration

Connected AI models to the chatbot for intelligent student support and assistance.

Mobile Responsiveness

The platform is fully responsive and optimized for mobile devices, ensuring students can access all features - from coding exercises to progress tracking - seamlessly on their smartphones and tablets.

Data Visualization

Screen Time Analytics

Interactive bar charts displaying student engagement metrics with smooth animations, helping track daily screen time and learning patterns.

Performance Results

Dynamic pie charts visualizing test results and completion rates with animated transitions, providing instant insights into student performance.

Theme Customization

Light Theme

A customizable interface with a light theme, ensuring a comfortable viewing experience in bright environments.

Dark Theme

The dark theme provides a consistent and eye-friendly experience across all features, ideal for low-light conditions.
