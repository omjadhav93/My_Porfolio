A smart public transport solution for real-time bus tracking, route optimization, and enhanced commuter convenience. This project was developed for the Smart India Hackathon (SIH) and qualified at the university level.

Project Overview

As the Backend Engineer for this project, I was responsible for architecting and building the server-side infrastructure. This included developing a sophisticated route-finding algorithm, creating APIs to power both the user-facing mobile app and the administrative dashboard, and ensuring real-time data processing for live bus tracking. Our solution provides passengers with accurate ETAs and route updates, while empowering administrators with a powerful fleet management tool.

Technology Stack

Node.js Express.js MongoDB Socket.IO React.js Dijkstra's Algorithm

Key Features

Real-Time Bus Tracking

Passengers can track bus locations live on a map, providing accurate ETAs and reducing waiting times at stops.

Advanced Route Search

A powerful search mechanism to find the most efficient routes, including transfers, between any two points in the city.

Fleet Management Dashboard

An admin dashboard providing a complete overview of fleet operations, including on-time performance and passenger load factors.

Route & Station Management

Admins can easily add, update, and manage bus routes and station details through a dedicated management interface.

Interactive Live Map

Users can view all nearby bus stops, explore routes visually, and get live updates directly on an interactive map interface.

Secure Authentication

A secure and simple OTP-based login system for passengers, ensuring quick and safe access to the platform.

Backend Implementation Highlights

Advanced Routing Algorithm

Implemented a custom route-searching mechanism using Dijkstra's Algorithm combined with a tree data structure to efficiently find the optimal path, including necessary bus transfers.

Real-Time Data Processing

Utilized WebSockets (Socket.IO) to create a persistent, low-latency connection for broadcasting live GPS coordinates from buses to multiple clients simultaneously.

RESTful APIs for Dashboard

Developed a comprehensive suite of RESTful APIs to feed aggregated data (ridership, on-time performance, fuel consumption) to the analytical admin dashboard.

Database Architecture

Designed and implemented MongoDB schemas for storing complex, interconnected data including routes, stations, bus schedules, and live tracking history efficiently.

Geolocation Services

Built backend logic to process raw GPS data, calculate ETAs based on real-time traffic and distance, and perform geospatial queries to find nearby stations.

Scalable Server Architecture

Configured a Node.js server with an event-driven, non-blocking architecture, ensuring high performance and scalability to handle thousands of concurrent users.

UI & Dashboard Showcase

Comprehensive Admin Dashboard

The admin interface provides a powerful, data-rich dashboard for monitoring all fleet operations and making informed, data-driven decisions.

Intuitive Mobile Interface

The platform is designed with a mobile-first approach, ensuring a clean, responsive, and seamless experience for passengers on the go.
